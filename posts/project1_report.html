  1. Abhijeet Chintakunta: App link

    • In the yearly bar graphs all the years were displayed so it was easy for the user to interpret

    • The title of the graph is reactive so if we change the stationname in the selection dropdown the stationname in the title changes this is done for all the graphs which helps the user to quickly interpret the data.

    • There is a dropdown called select the plottype in which we can select the plottype for a particular year so in this a select all option is included which enables the user to view all the graphs at the same time of that year in a single place

    • For the daily graphs the X axis represents date so displaying all 365 dates would make the graph cluttered so breakdown was introduced for every month in the X-axis.

    • In the interesting dates tab date is shown around the date to provide a quick insight for the user on what impact the date had on cta entries.
